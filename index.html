Nexus-crowd-plugin

This is a fork from sonatype/nexus-crowd-plugin to support up to date versions of nexus and crowd

Welcome to Nexus-crowd-plugin.

This fork is to keep an up to date version of the nexus-crowd-plugin. With so many updates both on the nexus and crowd front, it's hard to keep track! And those two awesome products work so well together....

Usage

  1. Add a nexus application to Crowd with a SECRET secret

  2. Create a file called crowd-plugin.xml under sonatype-work/nexus/conf

<?xml version="1.0" encoding="UTF-8"?> <crowdConfiguration> <crowdServerUrl>http://[crowdhost]:[crowdport]/crowd/</crowdServerUrl> <applicationName>nexus</applicationName> <applicationPassword>SECRET</applicationPassword> </crowdConfiguration>

  1. unzip the bundle under sonatype-work/nexus/plugin-repository

  2. Start Nexus and use the Administration->Server panel to add OSS Crowd Authentication Realm to the list of active realms.

Plugin Download

nexus-crowd-plugin-2.0.7-bundle.zip is the current release for Nexus 2.x and Crowd 2.x.
